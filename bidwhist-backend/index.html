BidWhist Backend Setup Guide

1. Install Java Development Kit (JDK)

  1. You need JDK 17 or newer for Spring Boot compatibility.

    2. ðŸ”— Download Temurin 21: AdoptOpenJDK

  2. After installation, verify:
    3. java -version

2. Clone the Git Repository

git clone https://github.com/your-username-or-org/bidwhist-backend.git
cd bidwhist-backend

3. Open the Project in VS Code

  1. File > Open Folder > Select "bidwhist-backend"

4. Install VS Code Extensions (One-Time)

  1. In VS Code, press Ctrl+Shift+X and install the following extensions:
  • Java Extension Pack
  • Spring Boot Extension Pack

5. Run the Spring Boot App

NOTE: The project includes a Maven Wrapper, so you don't need to install Maven globally.

  1. In VS Code's built-in terminal:
    • ./mvnw spring-boot:run (on macOS/Linux)
    • mvnw.cmd spring-boot:run (on Windows)
  2. Let Maven download dependencies (first time only). This may take a couple of minutes.
