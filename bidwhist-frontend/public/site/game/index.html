                      Getting Started

                      Use the following user guide to learn how to use the application and play Bid Whist!

                      Environment Setup

                      • Install Java (JDK 17+) Java JDK
                      • Install Maven and add bin to path Maven
                      • Install Node.js (Vers 18+) Node.js
                      • Build Project (Backend) mvn clean package
                      • Run Application (Backend) .\mvn spring-boot:run
                      • Install Dependencies (Frontend) cd bidwhist-frontend, then npm i
                      • Run Server (Frontend) npm run dev

                      Launching the Game

                      Once the backend and frontend services are running, open a browser and navigate to http://127.0.0.1:5173 or localhost:5173

                      Starting a Single Player Game

                      The application defaults to a single player game, as can be seen at the game's homepage

                      Figure 1: Single Player Welcome
                      • To start the game, enter a name in the textfield that says "Enter your name"
                      • Select a difficulty level [Easy, Medium, or Hard] from the drop-down menu
                      • Click Start Game

                      Starting a Multiplayer Game

                      Please note, multiplayer is only available through the website.

                      After loading the homepage for the Bid Whist App, find the slider on the welcome screen.

                      Click this slider to switch to the multiplayer home page.

                      • Like in Single Player mode: Enter your name
                      • Click Create Lobby

                      Joining a Multiplayer Game

                      To join a multiplayer game, an autogenerated 6-character Alphanumeric code is required.

                      This will be the same code that others will use to join the current lobby. Any player will have access to the code while waiting to join the game

                      Game Board Layout

                      Whether in Single Player or Multi-Player mode, the game will load and deal 12 cards to each player.

                      The game board contains 4 sets of cards (hands), a zone for bidding and playing cards, and a scoreboard. The bottom of the game board contains the user's hand. These are the cards the user can play each round.

                      When the game starts, the player must enter a bid (or how many hands the player believes they can win given the current hand). Place a number in the box under 'Place Your Bid' and press the Set Bid button. Alternatively, players can press the Pass or Quit.

                      The Pass Button registers a 0-bid to the game. Quit will quit the game.

                      Once all players input their bids, the round begins. The top of the Game board contains the scoreboard.

                      The scoreboard consists of 3 sections:

                      • Team A's Score
                      • Game Phase/Information (Team with the winning bid, Trump Suit and Direction, and Game Phase)
                      • Team B's Score

                      Player's Turn

                      On the player's turn, drag and drop a card from the player's hand. When starting to drag the card, a zone will appear on the board, indicating where the card should be dropped at.

                      Ending the round

                      The round ends after all players have played a card, and the team with the highest winning card, wins the hand (called a trick), and the round begins again. Won tricks are stored in the top left and bottom right of the game board.

                      • Bottom Right (1)—Team A's (Player's team) won tricks
                      • Top Left (2)—Team B's won tricks.

                      Play will continue until all cards have been played.

                      Next Round

                      Once all cards have been played, the round starts over with a new bidding phase. The scoreboard will update with team scores and information as the game progresses through each round.

                      Ending the Game

                      Players will continue to play through the rounds until one team reaches a score of 7.
